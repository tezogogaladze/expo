# expo

Interactive digital showcase for presenting products, spaces, or concepts in a structured and visually controlled environment.

---

## Overview

**expo** is a modular presentation system designed to simulate a digital showroom experience.

It allows users to explore curated content—products, environments, or visuals—through a clean, guided interface focused on clarity and visual impact.

---

## Features

- Structured showcase layout
- Section-based navigation
- Media-rich presentation (images, video, 3D)
- Minimal, distraction-free UI
- Responsive across devices

---

## Use Cases

- Furniture and interior showrooms
- Product catalogs
- Exhibition-style landing pages
- Brand presentations

---

## Tech Stack

- React / Next.js
- TypeScript
- Tailwind CSS

---

## Core Idea

Instead of traditional websites with scattered content, **expo** organizes information as a **controlled visual flow**.

Each section acts as a curated frame, allowing the viewer to focus on one piece of content at a time.
